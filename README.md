# piresc.dev

Personal portfolio and resume site for Pires Cerullo.

**Live at:** https://piresc.dev

## Stack

- **Frontend:** React + TypeScript
- **Build:** Vite
- **Styling:** Tailwind CSS
- **Hosting:** Cloudflare Pages

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

Automatically deploys to https://piresc.dev on every push to `main` via Cloudflare Pages.

## Adding CV

Place your CV as `public/piresc-cv.pdf` to make it downloadable from the site.
